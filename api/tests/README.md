# Тесты базы данных (DB Tests)

Тесты покрывают следующие сценарии:

* ✅ Сохранение результата предсказания (`save_to_db`)
* ✅ Корректность сохранённых данных
* ✅ Работа с несколькими записями
* ✅ Обновление `feedback`
* ✅ Обработка пустого списка пицц

## 🚀 Запуск тестов

Из корня проекта:

```bash
pytest -v
```
