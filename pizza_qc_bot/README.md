Pizza Quality Control Bot

Telegram-бот для автоматического распознавания видов пиццы и контроля качества. Бот принимает фотографии, отправляет их на внешний API с нейросетью и возвращает размеченное изображение с bounding boxes (рамками) и текстовым отчетом.

ОСНОВНЫЕ ВОЗМОЖНОСТИ:

-Детекция в реальном времени: Отправка фото в чат и получение мгновенного результата.

-Умная визуализация: Отрисовка рамок вокруг пицц с помощью библиотеки Pillow.

-Цветовая индикация (Светофор):

🟢 Зеленый: Высокая уверенность модели (> 80%).

🟡 Желтый: Средняя уверенность (50% - 80%).

🔴 Красный: Низкая уверенность (< 50%).

-Система обратной связи: Сбор фидбека (✅/❌) для улучшения точности модели (Active Learning).
  

СТЕК:

-Python 3.10+

-python-telegram-bot: Взаимодействие с API Telegram.

-aiohttp: Асинхронные HTTP-запросы к серверу нейросети.

-Pillow (PIL): Динамическая обработка изображений и рисование графики.

-python-dotenv: Безопасное хранение секретных ключей.



ДЛЯ ЗАПУСКА БОТА ВАМ ПОНАДОБЯТСЯ:

-Telegram Bot Token: Полученный у @BotFather.

-API ML-модели: Адрес развернутого сервера, принимающего изображения.

-API Key: Ключ авторизации для доступа к вашему API.

-Запуск:
```bash
python bot.py
```


СТРУКТУРА ПРОЕКТА:

- bot.py — Основной файл бота с логикой обработки сообщений.

- draw_boxes — Функция для отрисовки графики на изображениях.

- send_to_api — Модуль асинхронного взаимодействия с моделью.

- .env — Конфиденциальные настройки (не для GitHub).
