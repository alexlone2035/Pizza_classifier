Pizza Quality Control Bot

Telegram-бот для автоматического распознавания видов пиццы и контроля качества. Бот принимает фотографии, отправляет их на внешний API с нейросетью и возвращает размеченное изображение с bounding boxes (рамками) и текстовым отчетом.

Основные функции:

- Object Detection: Распознавание различных видов пиццы (Pepperoni, Margarita, Four Cheese и др.).

- Визуализация: Отрисовка цветных рамок вокруг найденных объектов прямо на фото с указанием уверенности модели.

- Интеграция с API: Асинхронное взаимодействие с ML-сервером через aiohttp.

- Система обратной связи: Возможность подтвердить или опровергнуть результат работы модели через встроенные кнопки (Inline Buttons).

  

Стек технологий:

- Язык: Python 3.9+

- Библиотека бота: python-telegram-bot

- Работа с изображениями: Pillow (PIL)

- Асинхронные запросы: aiohttp

- Управление окружением: python-dotenv



Для запуска бота вам понадобятся:

-Telegram Bot Token: Полученный у @BotFather.

-API ML-модели: Адрес развернутого сервера, принимающего изображения.

-API Key: Ключ авторизации для доступа к вашему API.



Установка и запуск:

-Клонируйте репозиторий:
```bash
git clone https://github.com/ваш_логин/название_проекта.git
cd название_проекта
```

-Установите зависимости:
```bash
pip install -r requirements.txt
```
Если файла requirements.txt еще нет, установите вручную:
```bash
pip install python-telegram-bot aiohttp Pillow python-dotenv
```

-Создайте файл .env в корневой папке и добавьте туда свои данные:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
API_BASE_URL=http://ваш_адрес_api:порт
API_KEY=ваш_секретный_ключ_авторизации
```

-Запустите бота:
```bash
python bot.py
```
